2.1 VCS_COMMON_VARAIBLES_MV

Description: This view presents common variables collected from the cost survey over time. Values are presented at the annual vessel-level for the year in which they were queried unless otherwise noted (nominal dollars). To be considered a common variable, the variable must appear in two or more cost surveys over time. Given that the survey instrument changed over time, there may be slight discrepancies in the variable over time. Please see the comments for further clarification.

Data warnings: Crew_Pymts in this year contains both crew and captain payments. Captain_Pymts is null but retained for the cost data application reasons.
